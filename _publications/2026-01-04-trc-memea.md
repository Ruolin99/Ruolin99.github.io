MemEA is a memory-enhanced adaptive motion planning framework for rule-compliant personalized autonomous driving in complex interactive traffic scenarios. The proposed method integrates a two-layer decision-planning architecture with a history memory retrieval module to generate regulation-abiding trajectories while adapting to individual user driving preferences and dynamic traffic interactions.

The framework addresses critical challenges in AV motion planning by unifying rigid traffic rule constraints, personalized driving demands and real-time computational efficiency. MemEA combines an upper rule-digitization decision layer for safety-compliant maneuver screening with a lower multi-objective trajectory optimization layer, plus a novel memory module that reuses historical planning cases to cut redundant online computation.

**Key contributions:**
- Bi-level adaptive planning architecture that balances traffic rule compliance and customized user driving preferences
- Memory-enhanced retrieval mechanism integrating environmental, numerical and semantic similarity matching for low-latency real-time planning
- Unified motion planning pipeline supporting both static legal regulations and ambiguous interactive traffic rules
- Consistent, scalable performance on mainstream public driving benchmarks with drastically reduced rule violations and computation latency

### Detailed Framework Overview & Empirical Validation

MemEA targets the research gap that current autonomous vehicle planners fail to simultaneously satisfy traffic rule obedience, personalized driving experience and fast real-time inference. It adopts a two-tier structure:
- **Upper decision layer:** Digitize static traffic laws and ambiguous interaction rules to filter out only legally allowed driving maneuvers as safety baselines;
- **Lower trajectory optimization layer:** Conduct multi-objective planning under rule constraints to produce trajectories matching user preference and real-time traffic context.

A core memory module is embedded to retrieve similar historical planning samples via three-dimensional similarity matching, which greatly lowers online reasoning overhead. Validated on CommonRoad (1,900 scenarios) and INTERACTION (1,200 scenarios):
- Overall traffic rule violations drop by **73%** vs. baseline planners;
- Decision layer alone cuts violations by **31.92%**; adaptive planning layer reduces collision rate by **10.08%**;
- Memory module slashes computing delay by **87.21%**, with stable performance across memory libraries of different scales.
