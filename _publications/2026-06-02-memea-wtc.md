![MemEA Framework](/images/memea.jpg)

MemEA is a memory-enhanced adaptive motion planning framework for rule-compliant personalized autonomous driving in complex interactive traffic scenarios. The proposed method integrates a two-layer decision-planning architecture with a history memory retrieval module to generate regulation-abiding trajectories while adapting to individual user driving preferences and dynamic traffic interactions.

**Key contributions:**
- Bi-level adaptive planning architecture that balances traffic rule compliance and customized user driving preferences
- Memory-enhanced retrieval mechanism integrating environmental, numerical and semantic similarity matching for low-latency real-time planning
- Unified motion planning pipeline supporting both static legal regulations and ambiguous interactive traffic rules
- Consistent, scalable performance on mainstream public driving benchmarks with drastically reduced rule violations and computation latency
